IDEA
====

IDEA clustering framework

## ENVIRONMENT
Python 3

## REQUIREMENT
scikit-learn (https://scikit-learn.org/stable/install.html)
```bash
pip install scikit-learn
```
PyMetis (https://pypi.org/project/PyMetis/)
```bash
pip install scikit-learn
pip install pybind11
pip install pymetis
```
